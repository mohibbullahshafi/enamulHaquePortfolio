# Personal Portfolio

A professional portfolio website with Node.js (Express) backend and React (Vite) frontend.

## Features
- Homepage
- About
- Projects
- Contact (with backend API)
- Courses (for future offerings)
- Consultancy Booking (for future extension)

## Getting Started

### 1. Backend
```
cd backend
npm install
npm run dev
```

### 2. Frontend
```
cd frontend
npm install
npm run dev
```

## GitHub Setup
1. Initialize git:
   ```
   git init
   git add .
   git commit -m "Initial commit: professional portfolio scaffold"
   ```
2. Create a new GitHub repo and push:
   ```
   git remote add origin https://github.com/yourusername/personal-portfolio.git
   git push -u origin main
   ```
